# truefix-config

Configuration parsing and models for TrueFix `SessionSettings` and FIX `.cfg` files. It covers
session defaults and overrides, schedules, dictionaries, stores, TLS, and connection settings.

Most applications use this through the `truefix` facade. See the
[configuration guide](https://github.com/truefix-labs/truefix/blob/main/docs/getting-started.md)
for examples and prerequisites.

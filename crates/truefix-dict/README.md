# truefix-dict

FIX data-dictionary loading, validation, and build-time typed-message code generation for TrueFix.
It provides both runtime `DataDictionary` validation and generated typed messages from the same
dictionary source.

The optional `dict-tooling` feature enables dictionary conversion tooling. See the
[TrueFix repository](https://github.com/truefix-labs/truefix) for supported dictionaries and commands.

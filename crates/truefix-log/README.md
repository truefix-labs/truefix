# truefix-log

TrueFix logging backends for screen, file, tracing, composite, and optional SQL, MSSQL, Redb, and
MongoDB sinks. Backends implement the public, object-safe logging interface used by the engine.

Most applications use this through `truefix`; see the
[TrueFix repository](https://github.com/truefix-labs/truefix) for configuration details.

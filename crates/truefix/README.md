# truefix

TrueFix is a Rust FIX engine facade providing the application trait, message cracking, and
re-exports of the core session, transport, configuration, dictionary, store, log, and binary crates.

## Installation

```toml
truefix = "0.1"
```

See the [TrueFix repository](https://github.com/truefix-labs/truefix) for installation, configuration,
and complete initiator/acceptor examples.
